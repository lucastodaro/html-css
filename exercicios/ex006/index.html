Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <B> (não semântica)

Nesta frase temos um termo em destaque usando a tag <STRONG> (com semântica)

Itálico / Ênfase

Nesta frase, temos um termo em itálico ,usando a tag <i> (não semântica)

Nesta frase, temos um termo em Ênfase, usando a tag <em> (com semântica)

Texto marcado

Podemos criar também um texto marcado com a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo, utlizando as tags <big> e <small>.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Utilizamos a tag <del>.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.Para isso, utilizamos a tag <ins>.

Texto sobreescrito

Para inserir textos como: x20+3, utilizamos <sup>.

Texto subescrito

Para inserir textos como: H20, utlizamos <sub>.
